# 📦 Amazon Sales Analysis Dashboard (Power BI)

## 🔹 Project Overview
This project presents an interactive Amazon Sales Analysis Dashboard built using Power BI. It provides insights into sales performance, profit trends, customer behavior, and product categories.

## 🔹 Objectives
- Analyze overall sales and profit performance
- Identify top-selling products and categories
- Track regional sales distribution
- Understand monthly and yearly trends

## 🔹 Tools & Technologies
- Power BI
- Excel (Dataset)

## 🔹 Key Insights
- Identified high-revenue product categories
- Analyzed profit vs sales relationship
- Discovered top-performing regions
- Visualized monthly sales trends

## 🔹 Dashboard Features
- Interactive filters (Region, Category, Date)
- KPI cards for Sales, Profit, Orders
- Bar charts, line charts, and pie charts
- Dynamic and user-friendly interface

## 🔹 Files Included
- Amazon_Sales_Dashboard.pbix
- Dataset (Excel/CSV)
- Dashboard Screenshots

## 🔹 Conclusion
This dashboard helps in making data-driven business decisions by providing clear and actionable insights into Amazon sales data.
